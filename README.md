# Desbytes Portfolio

Portfolio website — React + TypeScript SPA with i18n (EN/PL), contact form, and CI/CD.

**Live:** https://desbytes.com

---

## Repo structure

```
frontend/portfolio/    → main app (React + Vite)
frontend/              → legacy CRA (archived)
backend/               → Spring Boot skeleton
workers/contact/       → Cloudflare Worker (contact form API)
```

## Tech Stack

React 19, TypeScript, Vite, React Router, i18next, AWS EC2, Nginx, Cloudflare

For details see [frontend/portfolio/README.md](./frontend/portfolio/README.md).
